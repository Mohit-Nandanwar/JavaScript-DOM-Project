# JavaScript-DOM-Project
this tis the javascript document object model (dom) project, that is made in this project it show that how to interact with an webpage with the main role of javascript  or hoe to handel our booking service page. this page make more interactive with the help of css the basic html structure help us to make the body of the web page ...


This project is a simple web application that allows users to browse services, add them to a cart, and proceed with booking. It is designed with a user-friendly interface featuring:

A navigation bar with logo, links, username, and logout button.

A left section to display added items, total amount, and a booking form.

A right section to browse available services with images, names, prices, and action buttons.
